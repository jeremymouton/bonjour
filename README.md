# Bonjour

## Clone & Install

- `git clone https://github.com/jeremymouton/bonjour.git`
- `cd bonjour`

Install Gulp and Bower

- `npm install`
- `bower install`


### Running Gulp

- `gulp`

For auto-compile use `gulp watch`.


### Frontend notes

All assets go in `/src/` and are compiled with Gulp to the `/assets/` folder.
